# pong-name

## Features included:
```

> ✅ Player controls - Move left paddle with mouse position or arrow keys (↑/↓) 
> ✅ Computer AI - Right paddle automatically tracks the ball
> ✅ Bouncing ball - Physics with collision detection on walls and paddles
> ✅ Collision detection - Paddles and walls with ball spin mechanics
> ✅ Scoreboard - Displays player vs computer score
> ✅ Win condition - First to 11 points wins
> ✅ Reset button - Start a new game anytime
> ✅ Modern UI - Gradient background, smooth animations, and responsive design

```
